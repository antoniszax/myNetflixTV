# myNetflixTV

A Java application for managing movies and TV series, built with Swing. Includes login system, user roles (Admin/User), and full CRUD functionality.

## Features
- User & Admin login
- Add/Edit/Delete movies and shows
- User favorites
- Reviews & ratings
- GUI built entirely in Swing
- Data stored in binary files (object serialization) instead of a database

## How to Run
1. Open the project in IntelliJ
2. Run 'Main.java' file to do the set up
3. Run 'Login.java' file to start the application
4. Connect using one of the existent accounts (admin1 password1 , admin2 password2 , user1 password1 , user2 password2) or create new account and surf in the app.
5. When you terminate the program, if you run 'Login.java' again, you can connect with your new account.


## Tech Stack
- Java 17
- Swing GUI
- Object-oriented design
